# Requisitos do Sistema (Gestão de Clientes Agrícolas)

- Uma empresa tem funcionários que fazem login para gerenciar dados.
- Cada empresa pode ter vários agricultores (clientes).
- Cada agricultor pode ter várias fazendas.
- Cada fazenda pode ter vários talhões (fields).
- O gerenciamento de talhões é sazonal: em uma safra/temporada (ex.: 2025/1) a empresa pode gerenciar o talhão 1; em outra (ex.: 2025/2) pode gerenciar apenas o talhão 2.
- Arquivos podem ser gerados por talhão em diferentes safras (ligados à safra) ou apenas ao talhão (independentes da safra). Exemplo: imagens de satélite são do talhão, mas podem ser associadas a uma safra específica para quando o agricultor planta milho.
